## Overview

Election Frontend Project - A web application for managing and displaying election data, supporting multiple user roles (Admin, EC).

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Styling:** Tailwind CSS V4 + shadcn/ui
- **State Management:** Zustand + React Query
- **Forms:** React Hook Form + Zod
- **Charts & Maps:** Recharts, D3, React Simple Maps
- **Animation:** Framer Motion

## Core Features (Discovered)

- Multi-role Authentication & Selection Portal
- Dashboard & Data Visualization
- Form Validation
